## 单元测试 (UT)

### UT-01: 成本聚合查询（Rust）

| ID | 场景 | 输入 | 期望结果 |
|----|------|------|---------|
| UT-01.1 | 按模型分组 | 3 条不同 model 的 cost_records | 3 行结果，各有 tokens 和 cost 合计 |

| UT-01.4 | 按 Agent 分组 | 3 agents 各有数据 | Per-Agent 成本条形图数据正确 |

## 集成测试 (IT)

### IT-01: Dashboard 端到端

| 步骤 | 操作 | 期望结果 |
|------|------|---------|
| 1 | 完成 1 个 Mission（含 3 tasks） | 有历史数据 |
| 2 | 导航到 Insights/Dashboard | 4 个面板渲染 |
| 3 | 检查 Cost 面板 | 总成本、进度条、模型明细正确 |
| 4 | 检查 Quality 面板 | 平均分、分布图正确 |
| 5 | 检查 Efficiency 面板 | 利用率、甘特图正确 |
| 6 | 检查 Anomalies 面板 | 异常检测结果展示 |

### IT-02: Live 模式

| 步骤 | 操作 | 期望结果 |
|------|------|---------|
| 1 | 切换到 Live 模式 | 绿色 Live 脉冲点出现 |
| 2 | 启动一个 Mission | 成本数据实时递增 |
| 3 | 观察 3 秒 | 进度条和数字自动更新 |
| 4 | 切换到 History | 数据冻结、Live 点消失 |

### IT-03: 时间范围过滤

| 步骤 | 操作 | 期望结果 |
|------|------|---------|
| 1 | 选择 "Last 10m" | 仅显示最近 10 分钟数据 |
| 2 | 选择 "All" | 显示全量数据 |
| 3 | 对比两个范围 | 数据量和图表变化 |

### IT-04: 面板折叠

| 步骤 | 操作 | 期望结果 |
|------|------|---------|
| 1 | 点击 Cost 面板 header | Body 折叠、compact metrics 显示 |
